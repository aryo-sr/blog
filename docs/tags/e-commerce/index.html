Modul Shipping Ongkir Indonesia untuk Magento 2

July 10, 2019

magento e-commerce ongkir web
Selama mempelajari1 Magento, saya sudah membuat modul/extension ongkir (ongkos kirim) Indonesia untuk Magento 2. Modul ini mengambil data dari RajaOngkir yang menyediakan API cek ongkir dan tracking jasa kurir dan ekspedisi yang ada di Indonesia, termasuk JNE, J&T, Tiki, Pos, SiCepat, dll. Dengan modul ongkir ini, website Magento kita dapat menghitung ongkir barang berdasarkan kurir yang dipilih. Modul ini mirip dengan kode awal project yang saya kerjakan di tempat kerja saya –dulu.
